Welcome to .ods reader.

This is a python file written and checked in python 2.7

Until now I was converted my odses to xls in order to access the data through xlrd.

I accidentally went to: http://code.activestate.com/recipes/436066-read-openoffice-spreadsheet-as-list-of-lists-witho/

which helped me understood and create that python script.

In the near future I will add a python script to help those that cannot understand how to use it. And probably I will extend it to do more things...
